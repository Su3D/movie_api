Movie API

A Movie API to be built using the MERN (MongoDB, Express, React, and Node.js) stack and will allow users to access information about a variety of movies, directors, and genres.

Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET none JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user /movies/[title] GET none JSON object holding data about a single movie: description, genre, director, image URL, featured (y/n?) properties
Return a list of ALL genres to the user /genres GET none JSON object holding data about all the genres
Return data about a genre (description) by name/title (e.g., "Thriller") /genres/[type] GET none JSON object holding data about the genre: name/title, description
Return a list of ALL directors to the user /directors GET none JSON object holding data about all the directors
Return data about a director (bio, birth year, death year) by name /directors/[name] GET none JSON object holding data about the director: bio, birth year, death year
